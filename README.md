# Track

A personal learning-progress tracker built around three frameworks: **Marginal Gains (MG)**, **Kolb's Learning Cycle**, and **Spaced Interval Review (SIR)**.

## What it does

Log learning sessions and source material, reflect using Kolb's four-stage cycle, and schedule reviews at spaced intervals to retain what you've learned. Goals are organised hierarchically, streaks keep you accountable day-to-day, and multiple workspace slots let you separate distinct subjects or projects.

## Features

- **MG / Kolb / SIR tracker** — log sources, tag Kolb stages, set SIR review intervals (1 / 3 / 7 / 14 / 30 days)
- **Goals & streaks** — nested goal hierarchy with daily streak counter
- **Schedule** — day-view of planned and completed actions
- **Floating notes** — persistent notepad accessible from any page
- **Multi-slot workspaces** — isolated data per subject or project
- **Cloud sync** — optional Google sign-in via Firebase for cross-device access
- **Import / export** — per-slot JSON backup and restore

## Stack

HTML · CSS · Vanilla JS · React 18 (Babel CDN) · Tailwind CSS · Firebase (Auth + Firestore)

## Usage

Open `index.html` in a browser — no build step required.
